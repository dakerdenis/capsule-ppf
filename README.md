🛡 Capsule PPF — Paint Protection System
Full-featured Laravel-based web app for managing digital product verification, warranty generation, and service authentication for automotive paint protection films.

⚙️ Features
🔐 Digital Product Verification via QR code and unique code entry

📲 SMS Confirmation and warranty activation

📄 PDF Warranty Generator

🧑‍💼 Admin Panel for product/service management

🧰 Service Panel for authorized partners with access control

🌐 Multi-language Support (EN / RU)

🎨 Built with Laravel, Tailwind CSS, Vite

🚀 Tech Stack
Tech	Usage
Laravel	Backend / Routing
MySQL	Database
Tailwind CSS	Styling
Vite	Asset bundler
SMS API	Notifications & Warranty
DomPDF	PDF generation
Middleware	Role-based auth (Admin/Service)
